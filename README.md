# ShortLink

ShortLink - сокращатель ссылок на Java

## Stack

1. **Java: 21**
2. **Spring Boot**
3. **PostgreSQL**

## Реализация

**Реализивано:**

1. Получать сокращенную ссылку в ответе на POST-запрос, в теле которого отправляешь полную ссылку
2. При переходе по сокращенной ссылке перенаправлять на оригинальную ссылку, если ссылки оригинальной нет - отправить статус 404 с текстом ошибки
3. При повторной отправке оригинальной ссылки, которая уже есть, для ее сокращения, сервис не будет создавать новую ссылку, а отправит уже существующую

## Расширяемость

1. Сбор статистики по каждой ссылке
2. Кэширование новых и популярных ссылок
3. Ограничение частых запросов от одного пользователя
4. Добавление Frontend
