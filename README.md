# ShortLink

ShortLink - REST-сервис для сокращения URL с поддержкой кэширования, локализации и контейнерного запуска

## Stack

1. **Java: 21**
2. **Spring Boot**
3. **Docker/Docker Compose**
4. **PostgreSQL**
5. **Redis**

## Реализация

**Реализовано:**

1. Получать сокращенную ссылку в ответе на POST-запрос, в теле которого отправляешь полную ссылку
2. При переходе по сокращенной ссылке перенаправлять на оригинальную ссылку, если ссылки оригинальной нет - отправить
   статус 404 с текстом ошибки
3. При повторной отправке оригинальной ссылки, которая уже есть, для ее сокращения, сервис не будет создавать новую
   ссылку, а отправит уже существующую
4. Интернационализация (i18n) и локализация (l10n) сообщения на 2 языка: русский и английский
5. Кэширование новых ссылок (на 3 дня), а также политика для старых ссылок LFU (удаляются наименее используемые)

## Запуск

1. По примеру из `.env.example`, создать файл `.env`(`.local`,`.prod`) и заполнить данными
2. Запустить docker-compose командой:

```bash
docker compose --env-file .env.local \
   -f docker-compose.yml \
   -f docker-compose.local.yml \
   up --build
```

## Конфигурация

### Кэширование

Файл `redis.conf`:

```redis.conf
maxmemory 512mb # Максимальное количество ОЗУ для кэширования ссылок
maxmemory-policy allkeys-lfu # Политика удаления ключей
appendonly no # Отключение лога всех операций и сохранение на диск
```
